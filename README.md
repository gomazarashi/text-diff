# テキスト差分チェッカー

静的なテキスト差分チェックツールです。旧テキストと新テキストをブラウザ内で比較します。

公開URL:
https://gomazarashi.github.io/text-diff/

## 機能

- 行単位の差分表示
- 単語単位の差分表示
- 追加・削除・変更なしの色分け
- 入力内容の外部送信なし
- 入力内容の保存なし

## ローカル確認

任意の静的サーバーで確認できます。

```bash
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000/
```

## 長文比較の回帰検証

issue #1「対象テキストがある程度長くなるとページが動作しなくなる」の再発確認用に、手動実行の検証ページを用意しています。

```bash
python -m http.server 8000
```

ブラウザで以下を開き、`検証を実行` を押します。

```text
http://localhost:8000/tests/long-text-regression.html
```

検証ページでは、実ページを `iframe` で読み込み、次のケースを行単位比較で実行します。

- 1MB 程度の同一テキスト
- 1MB 程度のテキストで数行だけ差分
- 50,000 行の同一テキスト
- 改行の少ない 1MB 程度の長い 1 行

各ケースで `PASS` または `WARN` になり、ページ操作が戻ることを確認してください。`WARN` は処理に 10 秒以上かかった場合で、件数と結果 DOM の更新自体は完了しています。`FAIL` になった場合、またはブラウザタブが操作不能になった場合は回帰として扱います。

ヘッドレスブラウザ等から自動実行したい場合は、URL に `?autorun=1` を付けるとページ読み込み後に検証が開始されます。

## 注意

このツールは入力されたテキストをブラウザ内で処理します。入力内容はサーバーへ送信・保存されません。ただし、機密情報・個人情報・パスワード等の入力は避けてください。
